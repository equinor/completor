# Implementation of Completor® in FMU

[FMU](https://github.com/Equinor/fmu) or Fast Model Update is an integrated and automated workflow for predictive subsurface modelling and characterization.
This documentation will explain how to call Completor® through [ERT](https://github.com/equinor/ert). The workflow requires folder structure complying with FMU Standardization.
