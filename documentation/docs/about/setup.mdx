# Setup

If you work in Equinor, Completor® is available internally through the Komodo distribution system.

In addition, you can install Completor® by cloning the repository manually:
```shell
git clone https://github.com/equinor/completor.git
cd completor
python -m venv venv
source venv/bin/activate
pip install -e .
```
and to ensure it runs as intended, confirm it with:
```shell
completor --help
```
